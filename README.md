Voici la documentation faite par la section d'assault afin de pouvoir déployer l'application !
- Créer un dépôt

Va sur GitHub, clique sur New repository
Nom du dépôt :
mon-site (au choix)

Mets le dépôt en Public et ensuite clique sur create repository.
Dans le dépôt → Add file → Upload files

Upload :
index.html  Nomme le de cette manière (obligatoire)

style.css, script.js, images, etc.

Clique sur Commit changes.
Sinon tu peux créer le fichier directement sur Github et mettre ton code à l'intérieur.

- Activer GitHub Pages

Dans le dépôt → Settings, menu gauche → Pages
Section Source :

Branch : main

Folder : /root

Clique sur Save

Attends 30 secondes à 2 minutes.

Le lien de ton site va apparaitre et tu pourras y accéder.
